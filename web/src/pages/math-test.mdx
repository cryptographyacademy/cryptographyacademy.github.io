# Math Rendering Test

This page demonstrates KaTeX rendering of mathematical notation.

## Inline Math

Einstein's famous equation: $E = mc^2$

The Pythagorean theorem states that $a^2 + b^2 = c^2$ for right triangles.

Greek letters are commonly used: $\alpha$, $\beta$, $\gamma$, $\delta$, $\theta$, $\lambda$, $\sigma$, $\omega$.

## Display Math

### Summation

The sum of the first $n$ natural numbers:

$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$

### Integrals

The fundamental theorem of calculus:

$$\int_a^b f'(x) \, dx = f(b) - f(a)$$

### Fractions

A complex fraction:

$$\frac{1}{1 + \frac{1}{1 + \frac{1}{1 + x}}}$$

### Matrices

A 2x2 matrix:

$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$

A 3x3 identity matrix:

$$
I_3 = \begin{pmatrix}
1 & 0 & 0 \\
0 & 1 & 0 \\
0 & 0 & 1
\end{pmatrix}
$$

### Aligned Equations

Multiple equations aligned:

$$
\begin{aligned}
f(x) &= x^2 + 2x + 1 \\
&= (x + 1)^2
\end{aligned}
$$

## Cryptographic Notation

### Pedersen Commitment

The Pedersen commitment to message $m$ with randomness $r$ is:

$$C = g^m \cdot h^r$$

where $g, h \in \mathbb{G}$ are generators of a cyclic group of prime order $p$.

### Group Operations

For a group $(\mathbb{G}, \cdot)$ with generator $g$:

$$\mathbb{G} = \{g^0, g^1, g^2, \ldots, g^{p-1}\}$$

### Modular Arithmetic

Modular exponentiation:

$$a^b \equiv c \pmod{n}$$

### Elliptic Curves

An elliptic curve over a field $\mathbb{F}_p$:

$$E: y^2 = x^3 + ax + b$$

### Hashing

A collision-resistant hash function $H: \{0,1\}^* \to \{0,1\}^n$:

$$\Pr[H(x) = H(y) \land x \neq y] \leq \text{negl}(\lambda)$$

### Zero-Knowledge Proofs

A proof system $(\mathcal{P}, \mathcal{V})$ for language $L$:

$$\langle \mathcal{P}(w), \mathcal{V} \rangle(x) \in \{0, 1\}$$

### Finite Fields

Elements of a finite field:

$$\mathbb{F}_p = \{0, 1, 2, \ldots, p-1\}$$

where $p$ is prime.

### Polynomial Evaluation

For polynomial $f(X) = \sum_{i=0}^d a_i X^i$:

$$f(\alpha) = \sum_{i=0}^d a_i \alpha^i$$

### Set Notation

The set of all binary strings of length $n$:

$$\{0,1\}^n = \{x \in \{0,1\}^* : |x| = n\}$$

### Probability

Expected value of a random variable $X$:

$$\mathbb{E}[X] = \sum_{x} x \cdot \Pr[X = x]$$

## Complex Examples

### RSA Key Generation

1. Choose large primes $p, q$
2. Compute $N = pq$ and $\phi(N) = (p-1)(q-1)$
3. Choose $e$ such that $\gcd(e, \phi(N)) = 1$
4. Compute $d$ such that $ed \equiv 1 \pmod{\phi(N)}$

Public key: $(N, e)$, Private key: $(N, d)$

### Discrete Logarithm Problem

Given $g, h \in \mathbb{G}$ where $h = g^x$, find $x$:

$$\log_g(h) = x \text{ such that } g^x = h$$

This is believed to be hard in certain groups.
